About the dust measurement project

The dust measurement project is an initiative by some enthusiastic, passionate and convivial ERNI employees.

In the company-wide famous and prestigious annual Hack and Hicke event those employees met in an isolated location in the beautiful Swiss country site. There they commited in August 2018 to implement a sophisticated dust-measurement solution. The solution is based on latest state-of-the-art technology such as machine learning algorithms and 3-D printing. The participants self-orgnized into teams adressing following main topics:

  • Front-end
  • Back-end
  • Detector
  • Industrial Design (3D)
  • Data Science
  • Marketing

The resulting solution is showcased on this website. For more information please contact ERNI.
